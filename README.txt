README for Group 12 Database Project. 

Install MySQL
Install NODE.js
Change root account password to rootroot
Allow root account to have many concurent connections (1000)
Populate the database with DB_set_up.txt queries

This line might be required to be run in the database as it's own script:
  "  ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'rootroot'  "

Run app.js using NODE.js in the command line.
